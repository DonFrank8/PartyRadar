VIBEON Logo Variants
====================

This folder stores all provided logo variants from Google Drive for future brand usage.

Naming convention
-----------------
- `v1-*` = "Neues Logo Vers.1"
- `v2-*` = "Neues Logo Vers.2"

Current active logos used by the app
------------------------------------
- `../vibeon-logo-full.png` (website/header full logo)
- `../vibeon-logo-icon.png` (app/admin icon logo)

If you want to switch active logos
----------------------------------
1. Choose the desired files from this folder.
2. Replace:
   - `assets/vibeon-logo-full.png`
   - `assets/vibeon-logo-icon.png`
3. Commit and push.
